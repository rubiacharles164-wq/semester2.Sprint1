💾 DATA PERSISTENCE (LocalStorage)
═══════════════════════════════════════════════════════════════════════════════

Stored Data:
   
   KEY: "currentOrder"
   VALUE: Array of items in shopping cart
   {
       id: number,
       name: string,
       price: number,
       quantity: number
   }
   
   KEY: "orderHistory"
   VALUE: Array of completed orders
   {
       items: [...],
       customer: {...},
       total: number,
       orderDate: ISO string
   }

Benefits:
   ✓ Cart persists across page refreshes
   ✓ Items survive browser restart
   ✓ Order history for user reference
   ✓ No backend required
   ✓ Instant access to data

🚀 HOW TO USE
═══════════════════════════════════════════════════════════════════════════════

1. OPEN WEBSITE
   Open index.html in your web browser

2. BROWSE HOME PAGE
   • Read about the café
   • Watch reviews carousel (auto-rotates)
   • View location and contact info
   • Click "Explore Menu" button

3. BROWSE MENU
   • Click "Menu" in navigation
   • Filter by category or view all items
   • Click "Add to Order" on desired items
   • See confirmation message

4. PLACE ORDER
   • Click "Order Online" in navigation
   • Review your cart items
   • Click "Calculate Total" to compute costs
   • Fill in your information
   • Select delivery method
   • Click "Place Order"
   • See 2-second confirmation popup

5. VIEW CONFIRMATION
   • Order saved to localStorage
   • Form auto-clears
   • Ready for next order
